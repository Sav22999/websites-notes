Save data locally instead of sync with Firefox account
Since the version 3.3 the add-on automatically saves notes on your Firefox account, instead of save them locally. You can turn off the sync and restore the saving on local storage.
Choose the theme to use
If you choose Follow the Account theme this means the theme is not optimised for the add-on, although it will try to use the same theme correctly.
Avdanced managing of urls
When enabled you can save notes also per "sub-domains" like: https://example.page/a-page/*, https://example.page/a-page/other-page/*.
Show HTML text-formatting panel in the popup
When enabled you will see some buttons useful to insert text formatting (like bold, italic, etc.) in the popup – also you disable this, you will continue to be able to insert text-formatting via keyboard shortcuts.
Disable word-wrap notes text
Enable the language spellcheck detection for notes
Green addon icon when there are Global notes saved
When enabled the addon icon in the toolbar will be green if you previously saved Global notes.
Green addon icon when there are Domain notes saved
When enabled the addon icon in the toolbar will be green if you previously saved Domain notes for that domain.
Green addon icon when there are Page notes saved
When enabled the addon icon in the toolbar will be green if you previously saved Page notes for that page.
Green addon icon when there are Subdomain notes saved
When enabled the addon icon in the toolbar will be green if you previously saved Subdomain notes for that subdomain.
